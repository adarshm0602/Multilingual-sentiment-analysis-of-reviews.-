# Multilingual Sentiment Analysis of Reviews

A sentiment analysis project focused on analyzing reviews in Kannada and other languages using natural language processing and machine learning techniques.

## Project Structure

```
.
├── data/
│   ├── raw/          # Raw, unprocessed data
│   ├── processed/    # Cleaned and preprocessed data
│   └── models/       # Trained model files
├── src/
│   ├── preprocessing/  # Data cleaning and preprocessing scripts
│   ├── models/         # Model architecture and training scripts
│   └── utils/          # Utility functions and helpers
├── app/              # Streamlit web application
├── notebooks/        # Jupyter notebooks for exploration and analysis
└── tests/            # Unit tests
```

## Features

- Sentiment analysis for Kannada language reviews
- Multilingual support
- Web-based interface using Streamlit
- Preprocessing pipeline for regional languages
